Tailwind CSS Basic Utility Guide

Core Concepts

1. Styling with Utility Classes

Apply classes directly in HTML to style elements.

bg-blue-500 (background color), text-white (text color), font-bold (font weight), py-2 px-4 (vertical/horizontal padding), rounded-md (border-radius).

2. Hover, Focus, and Other States

Use prefixes to apply styles based on element states.

hover:bg-green-700, focus:ring, focus:border-blue-500.

3. Responsive Design

Apply different styles for various screen sizes using prefixes (e.g., sm:, md:, lg:).

Resize window to see text size change.

text-sm, sm:text-base, md:text-lg, lg:text-xl.

4. Dark Mode

Use `dark:` prefix for styles that apply in dark mode.

This content changes in Dark Mode (if enabled by system/config).

dark:bg-gray-900, dark:text-gray-100.

5. Colors

Access a consistent color palette for text, backgrounds, and borders.

Blue 500
Green 200
Border Red 400

bg-{color}-{shade}, text-{color}-{shade}, border-{color}-{shade}.

2. Spacing (Margin & Padding)

Control external and internal space around elements.

This div has p-6 (padding all sides) and m-4 (margin all sides).
This div has pt-8, pb-2, px-10 and mt-6.
This div is centered with mx-auto.

p-{size}, m-{size} for all sides. pt, pb, pl, pr for specific sides. px (x-axis), py (y-axis).

3. Sizing (Width & Height)

Control element dimensions using a consistent scale.

w-24 h-24
w-32 h-16
w-1/3 h-20

w-{size}, h-{size} for fixed sizes or fractions. w-full, h-auto, max-w-sm.
