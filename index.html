Tailwind CSS Utility Classes Cheatsheet

This cheatsheet explains key Tailwind CSS utility classes with examples, based on a card component design. Each class is demonstrated with a simple use case.

Card Container Classes

max-w-xs

Sets the maximum width to extra small (adjustable: max-w-sm, max-w-md).

Extra small width container

rounded-lg

Applies large rounded corners to the element.

Large rounded corners

overflow-hidden

Prevents child elements from spilling outside rounded corners.

shadow-lg

Applies a large box shadow for a lifting effect.

Large box shadow

bg-white

Sets the background color to white.

White background

Card Image Classes

w-full

Makes the element take the full width of its container.

Full-width element

h-48

Sets a fixed height for the element (12rem).

object-cover

Scales the image to fill the container, cropping if necessary.
